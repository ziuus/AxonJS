---
title: Agent
description: The central orchestrator that connects your LLM to your UI.
---

The `Agent` class is the heart of SynapseJS. It wraps your chosen LLM provider and gives it access to all 20 built-in tools (plus any custom tools you register). Using our **Feat System**, you can easily extend the Agent with complex behaviors like 3D character control (`CharacterFeat`) or advanced UI analysis.

### Constructor

```typescript
import { Agent } from '@synapsenodes/core';

const agent = new Agent({
  llmProvider: 'groq',       // 'groq' | 'gemini' | 'openai'
  apiKey: 'sk-...',           // your API key
  model: 'llama-3.3-70b-versatile', // optional
  systemPrompt: '...',       // optional custom system prompt
});
```

#### Options

| Option | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `llmProvider` | `'groq' \| 'gemini' \| 'openai'` | Yes | The LLM backend to use. |
| `apiKey` | `string` | Yes | API key for the chosen provider. |
| `model` | `string` | No | Model ID. Defaults to provider's recommended model. |
| `systemPrompt` | `string` | No | Override the default system prompt. |

### `agent.run()`

The main method. Processes a conversation and returns a text response plus any tool calls the LLM chose to emit.

```typescript
const result = await agent.run(messages, domElements);

// result.text     — the AI's text reply
// result.toolCalls — array of SynapseSignal objects to process

return Response.json({
  messages: [...messages, {
    role: 'assistant',
    content: result.text,
    toolCalls: result.toolCalls
  }]
});
```

### Extending with Feats

Feats are ready-to-use skill sets for your agent.

```typescript
import { Agent, CharacterFeat } from '@synapsenodes/core';

const agent = new Agent({ ... });
agent.use(CharacterFeat); // Immediately enables character gestures & control
```
